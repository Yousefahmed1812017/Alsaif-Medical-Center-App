Alsaif Medical Design System

A comprehensive, bilingual component library designed to build premium, modern, and accessible digital healthcare experiences.

Bilingual Architecture
This system uses Poppins for English (LTR) and Cairo for Arabic (RTL). Components automatically adapt icon placement and text alignment based on the active directionality.

Design System Rules

  • Radius: Keep components rounded with var(--radius-md) 10px or var(--radius-lg) 14px.
  • Shadows: Only use soft, large spread shadows to keep it premium and modern. Avoid harsh dark shadows.
  • Direction: Arabic layouts are strictly RTL utilizing Cairo font. Icons in buttons always precede text logically based on layout flow.
  • Palette: Stick rigidly to the 5 shades of blue provided for primary branding. Use Red/Green/Yellow only for semantic status alerts.
